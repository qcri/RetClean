# AI for DP

This is the official repository for the AI for Data Preparation project under QCRI. AI for DP is a tool that leverages language models for data preparation operations, such as imputation.

## Installation and Use

You must have node and npm installed on your maching. After downloading the repo, simply go to the diretory of the project and run -

```
npm install
```

After all the necessary modules are downloaded, launch the local web client with -

```
npm start
```
